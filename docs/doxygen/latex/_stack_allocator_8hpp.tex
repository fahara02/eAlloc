\doxysubsection{Detailed Description}
MCU/host-\/agnostic C++ allocator using a stack-\/allocated memory pool and TLSF. 

This allocator is suitable for embedded and host builds, provides minimal STL usage, and supports optional thread safety via \doxylink{classelock_1_1_i_lockable}{elock\+::\+ILockable}. It is intended for use with STL containers or custom allocation needs where a fixed-\/size stack buffer is desired.
