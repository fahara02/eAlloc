\doxysubsection{Detailed Description}
Portable bit manipulation intrinsics (ffs/fls) with architecture-\/specific optimizations. 

Provides fast find-\/first-\/set (ffs) and find-\/last-\/set (fls) implementations for GCC, MSVC, ARM, and generic fallback.


\begin{DoxyItemize}
\item Use dsa\+\_\+decl \doxylink{group__bitmanip_ga3e006b854f7c810a241a6ad9931a5a4c}{ffs(unsigned int)} and \doxylink{group__bitmanip_ga5b197bab96b83defaaacb5c7fa2abbc6}{fls(unsigned int)} for portable, inlined bit scanning.
\item Selects the best implementation for the detected compiler/architecture. 
\end{DoxyItemize}
