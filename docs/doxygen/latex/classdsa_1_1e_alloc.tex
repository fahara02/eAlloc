\doxysubsection{Detailed Description}
A memory allocator class based on the Two-\/\+Level Segregated Fit (\doxylink{classdsa_1_1_t_l_s_f}{TLSF}) algorithm. 

The \doxylink{classdsa_1_1e_alloc}{e\+Alloc} class provides efficient memory management with support for multiple pools, alignment, reallocation, and storage reporting.
